## 二、物理约束添加策略

### 当前约束

**波动方程残差 (L_wave)**：时间二阶导数平滑性

```
L_wave = mean(|u[t+1] - 2·u[t] + u[t-1]|²)
```

局限：仅为通用平滑正则化，未充分利用超声 NDT 领域的特有物理性质。

### 新增约束 1：频谱带限约束 (L_spectral) ⭐ 优先

**物理依据**：换能器频率响应有限，干净信号能量集中在中心频率附近，带外能量为噪声。

**数学形式**：

```
U(f) = FFT(u(t))
L_spectral = Σ_{f ∉ [f_low, f_high]} |U(f)|²  /  Σ |U(f)|²
```

**参考参数**：
- 中心频率：250 kHz
- 建议带宽范围：[100 kHz, 500 kHz]（-6dB 带宽）

**实现要点**：
- 使用 `torch.fft.rfft` 计算频谱
- 构建频率掩码，惩罚带外能量占比
- 不增加模型参数，仅影响损失函数

### 新增约束 2：因果性约束 (L_causal) ⭐ 推荐

**物理依据**：声波以有限速度传播，到达前信号应为零。

**数学形式**：

```
L_causal = mean(|u(t)|²),  t < t_arrival
```

**参考参数**：
- 钢材声速 5900 m/s，厚度 d → t_arrival = 2d/c
- 或直接取信号前 N 个采样点（如前 50 点）

**实现要点**：
- 对信号前 N 个时间步施加能量惩罚
- N 可设为超参数或根据声速和材料厚度计算

### 新增约束 3：包络平滑性 (L_envelope) — 可选

**物理依据**：超声回波的 Hilbert 包络应为平滑高斯形状。

```
envelope = |Hilbert(u(t))|
L_envelope = mean(|∂²envelope/∂t²|²)
```

**实现要点**：
- Hilbert 变换可通过 FFT 域实现
- 需注意梯度可传播性

### 最终复合损失函数

```
L_total = L_data + λ₁·L_wave + λ₂·L_spectral + λ₃·L_causal
```

**建议超参初始值**：

| 参数 | 初始值 | 调参范围 |
|------|--------|---------|
| λ₁ (wave) | 0.001 | [0.0001, 0.01] |
| λ₂ (spectral) | 0.01 | [0.001, 0.1] |
| λ₃ (causal) | 0.1 | [0.01, 1.0] |

### 实施路径

```
阶段 1: 实现 L_spectral → 单独验证效果
阶段 2: 实现 L_causal → 单独验证效果
阶段 3: 组合多约束 → 消融实验
阶段 4: 超参调优 → 最优配置
```
