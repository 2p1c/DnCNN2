## Key Files

### model/model.py (528 lines)

| Class | Lines | Purpose |
|-------|-------|---------|
| `DeepCAE` | ~180-350 | Base encoder-decoder CNN |
| `DeepCAE_PINN` | ~350-420 | DeepCAE with coordinate encoding |
| `DeepSetsPINN` | ~420-500 | DeepSets architecture with grid processing |
| `CoordinateMLP` | ~270-290 | Coordinate embedding (has embed_dim bug) |
| `GridEncoder` | ~290-360 | 2D grid encoding for DeepSets |

### scripts/train/train.py (1041 lines)

| Component | Lines | Purpose |
|-----------|-------|---------|
| `PINNLoss` | 148-158 | 1D damped wave equation residual |
| `DeepSetsPINNLoss` | 160-175 | 2D wave equation residual |
| `PINNDataset` | 70-120 | Data loading with physics grid generation |
| `DeepSetsDataset` | 120-145 | Data loading with 2D grid construction |
| `train_one_epoch` | 200-350 | Single epoch training loop |
| `validate` | 350-450 | Validation with RRMSE metric |
| `main` | 900-1041 | CLI entry point with argparser |

### scripts/transformer.py (~700 lines)

Converts .mat files to training directory structure:
- Reads noisy and clean .mat files
- Splits into train/val (default 80/20)
- Saves as .npy files in `data/train/{noisy,clean}/` and `data/val/{noisy,clean}/`

### scripts/run_unified_pipeline.py

End-to-end orchestrator combining transform + train + inference:
- `--skip_transform`: Use existing data
- `--skip_train`: Inference only mode
- `--checkpoint`: Load existing model
- `--log_experiment`: Create experiment records

### scripts/analysis/

| File | Purpose |
|------|---------|
| `inference.py` | PINN model inference, RRMSE computation |
| `inference_deepsets.py` | DeepSets model inference, batch processing |
| `acoustic_validation.py` | Acoustic physics validation |
| `preview_signals.py` | Signal visualization |

## Config Templates

- `pipeline_pinn_template.json`: PINN training config (epochs, batch_size, physics_weight, etc.)
- `pipeline_deepsets_template.json`: DeepSets training config
- `pipeline_tf_fusion_template.json`: TensorFlow fusion experiment config

## Entry Points

```bash
# Data preparation
uv run python scripts/transformer.py --noisy data/noisy.mat --clean data/clean.mat --output data

# Training
uv run python scripts/train/train.py --pipeline pinn --mode file --data_path data --epochs 100

# Unified pipeline
uv run python scripts/run_unified_pipeline.py --config configs/pipeline_pinn_template.json

# Inference
uv run python scripts/analysis/inference.py --input data/noisy.mat --output results/ --checkpoint <path>
```

## Data Format

- **Input**: .mat files (MATLAB format) containing signal arrays
- **Intermediate**: .npy files in train/val directories
- **Output**: .mat files with denoised signals and RRMSE metrics
